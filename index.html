            データのバックアップ

            写真以外の記録をファイルに保存できます。機種変更はGoogleログインの同期で引き継がれるので、これは万一の保険です。

            AIによる料理ジャンル判定

            Anthropic APIキーを設定すると、登録時に料理写真をAI（Claude）が解析し、料理ジャンル・店舗ジャンルを自動判定します。 キーはこのブラウザ内に保存され、ログイン中はアカウントにも控えが保存されます（消えても自動復元）。 キーは platform.claude.com で取得できます。

            店舗検索の強化（Googleマップ）

            Google MapsのAPIキーを設定すると、店名検索にGoogleマップのデータが加わり、地図データに載っていない個人店も見つかります。 Google Cloud Console で「Places API (New)」を有効にして取得します（無料枠あり）。
