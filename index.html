            ⚙️ 設定

            AIによる料理ジャンル判定

            Anthropic APIキーを設定すると、登録時に料理写真をAI（Claude）が解析し、 料理ジャンル・店舗ジャンルを自動判定します。
            キーはこのブラウザ内（localStorage）にのみ保存され、写真はAnthropic APIへ直接送信されます。 キーは platform.claude.com で取得できます。

            店舗検索の強化（Googleマップ）

            Google MapsのAPIキーを設定すると、店名検索にGoogleマップのデータが加わり、 無料の地図データに載っていない個人店もほぼ確実に見つかるようになります。 キーは Google Cloud Console で「Places API (New)」を有効にして取得します（無料枠あり）。
